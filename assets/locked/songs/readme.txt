Put any songs you want to have as unlockable here. **Both folders listed below must be made in the same folder as this file!**

# Locking song charts
To lock a song chart, move the json files in the data folder for your song (excluding events) to a new folder called `your-song/data`.

# Locking song audio
To lock the audio for your song, move the ogg files in the folder for your song to a new folder called `your-song/audio`.
